Switch theme of App
===================

- To change the Resources on the fly (dynamic Loading)

- Create a single CSS file for each theme (Red, Green, Blue). You should edit CSS files not to conflict each other. 
  eg: <link rel="stylesheet" ng-href="{{myTheme}}.css">


Approaches
----------
 - Local Storage
 - ng-href
