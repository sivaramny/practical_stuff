1. Download the Angular file from the below

CDN : https://cdnjs.com/libraries/angular.js/

Bower : bower install angular

Angular core: https://code.angularjs.org/

2. Included the AngularJS JavaScript file in the HTML page 'head' or 'body' tag

<script src = "http://ajax.googleapis.com/ajax/libs/angularjs//angular.min.js"></script>

3. Include 'ng-app' to bootstrap angular into our app

<script src = "http://ajax.googleapis.com/ajax/libs/angularjs//angular.min.js"></script>
