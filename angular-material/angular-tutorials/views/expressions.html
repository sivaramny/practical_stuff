• Expressions are used to bind application data to html.
  • Expressions are written inside double braces: {{  }}
    eg:{{5+5}}
  • Expressions binds data to HTML the same way as the ng-bind directive.
  • Expressions support filters also.
<html>
<head>
<title>Expressions Demo1</title>
</head>

<body>
<div ng-app = "" ng-init = "person = {name:'Sivaram' };">
<p>Hello { {person.name} }!</p>
</div>
